Jogamos o resultado da execução desse comando dentro de uma variável:


        var n1 = window.prompt('Digite um número: ')
        var n2 = window.prompt('Digite outro número: ')
        var s = n1 + n2
        window.alert('A soma dos valores é ' + s)

Acima, o sinal de soma + terá o efeito de concantenação, não irá somar um número mais o outro e sim unir um número ao outro. Por exemplo se colocarmos 6 e depois 2 no final irá aparecer no meu alerta 62.

O + no JavaScript serve tanto para adição quanto para concatenação, precisamos definir o tipo de valor que temos para o javascript entender qual das duas eles deve fazer, se será soma ou concatenação.

//(number + number) para adição
//(string + string) para concatenação

Porém o prompt tem como característica retornar o valor sempre como uma string, mesmo que eu digite um número ele trata como uma string. Precisamos converter o n1 e n2 de string para número. Para essa conversão temos várias maneiras, podemos usar qualquer umas das seguintes:

O primeiro é para converter para um número inteiro, o segundo para um número real (com vírgula). Importante que as letras em maiúsculo e minúsculo são exatamente da forma mostrada (Case sensitive).

Eu posso converter os valores direto do prompt, antes de somar:


        var n1 = Number.parseInt(window.prompt('Digite um número: ')) //conversão
        var n2 = Number.parseInt(window.prompt('Digite outro número: ')) //conversão
        var s = n1 + n2
        window.alert('A soma dos valores é ' + s)

Precisávamos saber se iamos usar um número inteiro ou um número real. Essa era uma forma antiga de converter, agora só precisamos usar Number(n), porém não elimina as de cima (podemos usar quando precisamos definir só número inteiros ou só reais por exemplo):

Podemos converter um número em uma string também com String(n)-> o n é o que eu quero converter, o S é maiúsculo:


        var n1 = Number(window.prompt('Digite um número: ')) //conversão
        var n2 = Number(window.prompt('Digite outro número: ')) //conversão
        var s = n1 + n2
        window.alert('A soma dos valores é ' + String(s))

        var n1 = Number(window.prompt('Digite um número: ')) //conversão
