Eventos é tudo aquilo que pode acontencer com qualquer elemento, no nosso caso é a div acima.

Podem acontecer divervas coisas, o mais comum são eventos de mouse. Como o mouseenter (significa que o mouse entrou na div), o mousemove(movimentos com o mouse dentro da div), mousedown(quando eu clico e seguro com o botão do mouse), moseup(quando eu solto o botão do mouse dispara esse evento), click(apertar e soltar o botão), mouseout(quando movo o mouse pra fora da div).

Existem outros eventos, veja no google: javascript dom event list ou veja no site da mozilla aqui

Para dispara um evento precisamos saber o que é uma função em javascript.

Função

Uma função é um conjunto de códigos/linhas que serão executadas só quando o evento ocorrer. Por exemplo esse código só vai ocorrer quando eu clicar dentro da minha div.

Pra executar várias linhas, somente quando o evento ocorrer, precisamos coloca-las em um bloco (conjunto de várias linhas).

Depois precisamos nomear esse bloco como uma function. No javascript existe a chamada função anônima(uma função sem nome). Porem para que o método funcione precisamos dar nomes para essas funções e geralmente os nomes das funções são ações que podemos realizar. Dentro dos parêntes colocamos os parâmetros.

Os eventos podem ser configurados na parte HTML ou direto no javascript. Para dispara uma ação colocamos on antes do nome da ação, que no caso abaixo é click. O onclick vai chamar o nome da ação que queremos, por isso digitamos a ação que queremos logo em seguida.
